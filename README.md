# CV Optimizer

A MERN app to optimize CVs using AI.

## Features
- Upload PDF CV.
- Input job description.
- AI analysis with suggestions and scoring using free OpenRouter models.

## Setup
1. Clone repo.
2. Setup backend and frontend as above.
3. Get OpenRouter API key (free tier).
4. Run backend and frontend.

## Limitations
- Free LLMs may have rate limits; handle errors.
- PDF must be text-based (not scanned images).
- Extend with auth, history storage.

## Future Improvements
- Add CV marking visualization (e.g., highlighted PDF).
- Use multiple LLMs in parallel for better results.
- Deploy to cloud.

For questions, refer to OpenRouter docs: https://openrouter.ai/docs
